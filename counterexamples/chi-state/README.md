# chi-state loophole

## Minimal Model

Status: Counterexample candidate. Introduce one explicit orbital-timescale internal coordinate `chi_A(tau)`:

```math
S_A
=
\int d\tau \left[
-m_0 \left(1 + s Y(z_A)\right)
+ \frac{1}{2} \dot{\chi}_A^2
- \frac{1}{2} \omega_\chi^2 \chi_A^2
+ g_\chi \chi_A Y(z_A)
\right],
```

with `omega_chi ~ Omega_orb`.

## Why It Escapes Theorem A

- Status: Counterexample candidate. The Euler-Lagrange equation

```math
\ddot{\chi}_A + \omega_\chi^2 \chi_A = g_\chi Y(z_A)
```

shows that the free-fall response depends on the initial conditions of `chi_A` as well as on the instantaneous external field.
- Status: Counterexample candidate. That dependence cannot be compressed into a finite set of instantaneous sensitivity coordinates unless `chi_A` is heavy enough to be adiabatically eliminated.
- Status: Counterexample candidate. The violated theorem assumption is A4, not locality or analyticity.

## Smallest-Model Reason

- Status: Counterexample candidate. Only one new degree of freedom is added, and the rest of the worldline EFT remains local and analytic.
