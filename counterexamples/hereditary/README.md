# hereditary loophole

## Minimal Model

Status: Counterexample candidate. Replace the local monopole action by a retarded kernel:

```math
S_{\rm mem}
=
\frac{\lambda_h}{2}
\int d\tau \, d\tau'
\exp\!\left(-\frac{|\tau - \tau'|}{T_h}\right)
Y(\tau) Y(\tau').
```

## Why It Escapes Theorem A

- Status: Counterexample candidate. The force depends on the past history of `Y`, not only on its instantaneous value and a finite Taylor jet.
- Status: Counterexample candidate. Even if the kernel is analytic in each argument, the resulting effective action is not local on the worldline.
- Status: Counterexample candidate. The violated theorem assumption is A3.
