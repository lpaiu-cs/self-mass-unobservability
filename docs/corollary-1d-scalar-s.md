# Corollary: One-Dimensional Scalar s

- Status: Conjectural. If the free-fall monopole response manifold is rank one over the chosen observable family, then there exists a coordinate `sigma` such that

```math
m_A(Y) = m_A^{(0)} \left[ 1 + s_A \sigma(Y) + O(\sigma^2) \right].
```

- Status: Conjectural. In that special case, `s_A` is not a new principle; it is the one-coordinate projection of the higher-dimensional sensitivity manifold from Theorem A.
- Status: Counterexample candidate. The corollary fails immediately if two independent invariant directions survive at the same observable order, or if the response is nonanalytic or history dependent.
