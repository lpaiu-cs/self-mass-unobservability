## Abstract Skeleton

Status: Conjectural. The abstract should make three boundaries explicit:

```text
We study the assumption that compact-body free-fall deviations contain no
orbital-timescale internal state variable. A first-order relaxation state
produces phase lag and a shared pole, but under the current orbital and
two-tone dictionaries it remains underbudget against finite shared static
comparators. We then analyze a second-order internal mode. Its quadratic
denominator survives finite shared free-fall-style projection, and exact-in-e
eccentric forcing supplies parameterized resonance-aware detectability
regions. With minimal nonlinear drive/readout, generated harmonics can carry
the same denominator, exceed finite static nonlinear comparator budgets, and
remain locally separable from the linear response in a non-fine-tuned
dimensionless parameter region. The result is not an instrument forecast; it
is a theorem-level counterexample candidate to the no-internal-state
assumption.
```

## Main Theorem Statements To Promote

Status: Proven. One-state control theorem: a finite set of one-state
relaxation lines or sidebands is not novel unless it exceeds the finite
shared linear/static nonlinear comparator and projection nuisance budgets.

Status: Proven. One-state current-dictionary no-go: the current orbital
`n,2n -> 3n` and two-tone dictionaries are underbudget against realistic
first-derivative or first-degree static nonlinear comparators.

Status: Counterexample candidate. Second-order projection theorem: finite
shared acceleration-like or range-like projection nuisance preserves the
internal quadratic denominator, while arbitrary per-frequency projection
nuisance collapses the observable claim.

Status: Counterexample candidate. Linear second-order detectability theorem:
with enough usable exact-in-e harmonics to exceed `N+2+K_Lambda` and bracket
the phase-wrap location, the line shape can be budget-breaking.

Status: Counterexample candidate. Nonlinear second-order generated-line
theorem: generated harmonics can carry the same denominator and can beat a
degree-`M` static nonlinear comparator only after at least `M+2+K_Lambda`
usable generated samples.

Status: Counterexample candidate. Component-separability theorem: the
generated amplitude `Q_beta` must add local Jacobian rank relative to the
shared linear and projection nuisance model.

Status: Proven. Analytic phase-boundary theorem: the nonlinear positive
region is governed by count, bracket, amplitude-usability, and component-rank
boundaries, and those boundaries reproduce the default robustness-grid
verdicts.
