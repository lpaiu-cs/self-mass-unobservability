# Content
Full release associated with [Explanation of the exceptionally strong timing noise of PSR J0337+1715 by a circum-ternary planet and consequences for gravity tests, Voisin, Cognard, Saillenfest, Tauris, Wex, Guillemot, Theureau, Freire, Kramer, A&A 2024][1]. 

This release can be dowloaded from <https://doi.org/10.5281/zenodo.13899771>


- `Analysis/` : Main results, and scripts to producess them. 
- `Data/` : times of arrival of PSR J0337+1715 used in this study
- `nutimo/`: source code of *Nutimo*, timing analysis software used to analyse the data. 
- `Paper/` : preprint version (including latex source) of the article
